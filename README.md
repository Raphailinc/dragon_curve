# Dragon Curve Playground

Современная демо кривой дракона: интерактивные ползунки итераций/скорости, выбор направления, цвета и толщины линии, а также экспорт в SVG.

## Запуск
Просто откройте `index.html` в браузере. Никаких зависимостей.

## Управление
- Итерации (5–18): глубина рекурсии.
- Скорость: задержка между отрисовками сегментов (анимация).
- Направление: начальный вектор (вправо/влево/вверх/вниз).
- Цвет и толщина линии.
- Кнопки «Сброс» и «Скачать SVG».

## Файлы
- `index.html` — разметка и подключение.
- `style.css` — тёмная тема и сетка расположения.
- `main.js` — генерация точек кривой, масштабирование в SVG, анимация и экспорт.

Алгоритм: рекурсивное деление отрезка с поворотом на 90°, итоговые точки масштабируются под размер SVG, и рисование идёт поэтапно для визуализации роста кривой.
